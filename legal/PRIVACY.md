# プライバシーポリシー / Privacy Policy

## 日本語

### 1. 基本方針
「やさしい画面 (calm-screen)」(以下、本拡張機能) は、利用者のプライバシーを最大限に尊重します。
本拡張機能は、個人を識別できる情報を一切収集せず、外部サーバーへも送信しません。

### 2. 収集しない情報
本拡張機能は以下の情報を収集しません。
- 氏名、メールアドレス、電話番号などの個人情報
- 閲覧履歴、検索履歴、Cookie
- IP アドレス、位置情報、デバイス識別子
- 入力フォームの内容、アカウント情報
- 利用統計、行動ログ、クリック履歴

### 3. ローカル保存するデータ
本拡張機能は機能提供のために以下のデータを、利用者のブラウザ内 (chrome.storage.local) にのみ保存します。
これらは外部に送信されることはありません。
- 各機能 (青色光カット、彩度抑制、アニメ抑制、ダークモード強制、輝度上限) の有効/無効状態
- Premium 機能の解放状態、無料お試し開始タイムスタンプ
- 利用者の設定値 (強度、対象範囲など)

利用者は Chrome の拡張機能管理画面から本拡張機能をアンインストールすることで、上記データを完全に削除できます。

### 4. 外部サービスとの通信
本拡張機能は、通常動作時に外部サーバーと通信しません。
ただし Premium 機能の購入時にのみ、決済処理のために Stripe Checkout (Stripe, Inc.) へ遷移します。
Stripe のプライバシーポリシーは https://stripe.com/privacy をご確認ください。

### 5. 広告・トラッキング
本拡張機能は広告を一切表示せず、第三者のトラッキングコードも一切含みません。

### 6. 子供のプライバシー
本拡張機能は子供を含む全年齢の利用を想定しており、いかなる年齢の利用者からも個人情報を収集しません。

### 7. ポリシーの変更
本ポリシーが変更された場合、本ファイル冒頭の「最終更新日」を更新します。重大な変更時は拡張機能のアップデート通知でお知らせします。

### 8. お問い合わせ
本ポリシーに関するご質問は、GitHub Issues (https://github.com/tabisurushosai/calm-screen/issues) にてお願いいたします。

## English

### 1. Basic Policy
"calm-screen" (the "Extension") respects user privacy to the fullest extent.
The Extension does not collect any personally identifiable information, nor does it transmit any data to external servers.

### 2. Information Not Collected
The Extension does not collect the following:
- Personal information such as name, email address, or phone number
- Browsing history, search history, or cookies
- IP address, location, or device identifiers
- Form input or account credentials
- Usage statistics, behavior logs, or click history

### 3. Data Stored Locally
For functionality, the Extension stores the following data only within the user's browser via chrome.storage.local. This data is never transmitted externally:
- Enabled/disabled state of each feature (blue light filter, desaturation, animation mute, dark mode force, brightness cap)
- Premium unlock state and free trial start timestamp
- User preferences (intensity, scope, etc.)

Uninstalling the Extension from Chrome's extension management page deletes all of the above data.

### 4. External Communication
The Extension does not communicate with external servers during normal operation.
The only exception is Premium purchase, which redirects to Stripe Checkout (Stripe, Inc.) for payment processing.
See Stripe's privacy policy at https://stripe.com/privacy for details.

### 5. Advertising and Tracking
The Extension contains no advertising and no third-party tracking code.

### 6. Children's Privacy
The Extension is designed for users of all ages, including children, and does not collect personal information from any user regardless of age.

### 7. Policy Changes
If this policy changes, the "Last updated" date at the top of this file will be updated. Significant changes will be announced via extension update notifications.

### 8. Contact
For questions about this policy, please open an issue at https://github.com/tabisurushosai/calm-screen/issues.
